## Description
A personal portfolio website powered by [preact](https://preactjs.com/). 

Visit at [https://arkadyt.com](https://arkadyt.com) or [www.arkadyt.com](https://arkadyt.com).

## To run locally
```
git clone https://github.com/arkadyt/arkadyt.com.git
cd ./arkadyt.com
npm install
npm start
```
